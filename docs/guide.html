Markmodern Syntax Guide

A comprehensive guide to all syntax features and their display examples.

Headings

Syntax:

# h1
## h2
### h3
#### h4
##### h5
###### h6

Display:

h1

h2

h3

h4

h5
h6

Text Formatting

Syntax:

**bold**
*italic*
***bold and italic***
~~strikethrough~~
`code`

Display:

bold

italic

bold and italic

strikethrough

code

Links

Syntax:

[link](https://example.com)

Display:

link

Syntax:

![alt text](https://via.placeholder.com/300x200)
-![with caption](https://via.placeholder.com/400x300)

Display:

Lists

Syntax:

- item 1
- item 2
  - nested 1
  - nested 2
- item 3

Display:

  • item 1
  • item 2
    • nested 1
    • nested 2
  • item 3

Ordered:

1. first
2. second
   1. nested
3. third

Display:

  1. first
  2. second
    1. nested
  3. third

Checkboxes:

- [x] completed
- [ ] incomplete

Display:

  • ☑ completed
  • ☐ incomplete

Code Blocks

Syntax:

```javascript
const x = 10;
console.log(x);
```

Display:

const x = 10;
console.log(x);

Blockquotes

Syntax:

> This is a quote
> Multiple lines work

Display:

This is a quote
Multiple lines work

Syntax:

Syntax:

===Click to expand

Admonition Block

Syntax:

:::note info
:::note warn
:::note alert

Display:

This is an information message.

This is a warning message.

This is a critical warning.
